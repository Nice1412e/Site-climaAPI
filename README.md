# Site-climaAPI
# 🌦️ Clima Tempo Web

Aplicação web desenvolvida com o objetivo de exibir informações meteorológicas em tempo real a partir da busca por cidades. O projeto consome dados de uma API externa e apresenta, de forma simples e intuitiva, as condições climáticas atuais.

---

## 🚀 Funcionalidades

* 🔍 Busca de clima por cidade
* 🌡️ Exibição da temperatura em graus Celsius
* ☁️ Descrição do clima (ex: nublado, ensolarado, chuva)
* 📍 Nome da cidade pesquisada
* ❌ Tratamento de erro para cidades inválidas

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* API de clima (OpenWeatherMap)

---

## 🌐 Como funciona

O usuário insere o nome de uma cidade no campo de busca. A aplicação realiza uma requisição HTTP para a API de clima e retorna os dados em tempo real, exibindo as principais informações na tela.
